# NameGuard JS

## Running Tests

1. Set the environment variables documented in `.example.env`.

2. Build the NameGuard SDK.

    ```bash
    cd /path/to/namekit/packages/nameguard-sdk
    pnpm build
    ```

3. Install NameGuard JS dependencies.

    ```bash
    cd /path/to/namekit/packages/nameguard-js
    pnpm install
    ```

4. Run the tests.

    ```bash
    pnpm test
    ```
